# Task: Brunton taxonomy reconstruction

## Skill under test

- `related-work-synthesis`
- `scientific-journal-writing`

## Gold reference

Steven L. Brunton, Bernd R. Noack, Petros Koumoutsakos, *Machine Learning for Fluid Mechanics*, Annual Review of Fluid Mechanics, 2020.

## Dumb-agent prompt

Build a related-work taxonomy for a new CFD-AI paper using Brunton-style field mapping. Include where ML enters the fluid-mechanics workflow and how a new contribution should position itself.

## Expected answer-key patterns

- Taxonomy over chronology.
- Categories like reduced-order modeling, flow control, turbulence closure, sensing/reconstruction, optimization, discovery.
- Emphasis on scientific insight and workflow role, not just model performance.
- Clear warning against black-box claims without physical interpretation.
