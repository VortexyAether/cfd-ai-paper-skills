# Figure Evidence Rubric

Score each figure/table as evidence, not decoration.

| Score | Descriptor | Test |
|---|---|---|
| 0 | Decorative | Shows fields/schematics but supports no claim. |
| 1 | Weak evidence | Claim is implied but missing scale, metric, or comparator. |
| 2 | Usable evidence | Supports one claim with clear labels, scale, and metric. |
| 3 | Strong evidence | Supports a claim with comparator, uncertainty/error, physical diagnostic, and limitation where relevant. |

## Required checks

| Check | Pass condition |
|---|---|
| Claim link | Caption names the claim or takeaway. |
| CFD labels | Variables, units/nondimensionalization, Re/regime, BC/geometry context where needed. |
| Comparability | Same color scale and aligned panels for GT/pred/error fields. |
| Quantification | Caption or panel gives metric, not only qualitative impression. |
| Physical diagnostic | Turbulence/flow claim has spectrum, residual, force, conservation, pressure drop, heat flux, or other relevant diagnostic. |
| Failure boundary | Generalization/robustness claim includes failure or OOD evidence. |
